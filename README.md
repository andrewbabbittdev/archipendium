## What is Archipendium?
Archipendium is a plugin for giving hints for your Archipelago games.

## What is in this repo?
This repository contains all source code for Archipendium. Everything needed to develop or deploy Archipendium is self contained in this repository.

## Getting started
To start developing Archipendium you need just a few things:

- [.NET 10 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)
- [Final Fantasy XIV](https://na.finalfantasyxiv.com/)
- [Dalamud](https://dalamud.dev/faq/getting-started/)

## Run Archipendium
Once you have all the dependencies listed above load the plugin in Dalamud and you can access the following commands:

- `/ap` - Opens the UI. Additional text will send a message if connected.
